# Posmotry-v-okno (CSS Styling Project)

## Описание проекта

Это веб-приложение отображает коллекцию видеороликов с возможностью поиска и подгрузки дополнительных видео. Основной функционал (загрузка данных, переключение видео, поиск) уже реализован в скрипте.  

В рамках этой проектной работы я **дополнил и улучшил CSS** для интерфейса:  

- Стилизовал форму поиска и текстовые поля.  
- Сделал аккуратные hover, focus-visible и active состояния для ссылок, кнопок и чекбоксов.  
- Настроил визуальное отображение выбранных и активных элементов.  
- Сохранил все интерактивные состояния приложения (прелоадеры, ошибки, выбор видео).  

## Используемые технологии

- HTML5 (готовая разметка)  
- CSS3 (Flexbox, modern selectors, focus-visible, :has(), box-shadow)  
- JavaScript (готовый скрипт для работы приложения)
